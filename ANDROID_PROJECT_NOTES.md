# UniSync Android Prototype

This is a draft Android Studio project for a MOBDEVE student productivity app.

## Stack

- Kotlin
- XML layouts
- Material Design 3
- ConstraintLayout
- RecyclerView
- Bottom Navigation

## Main Flow

1. Splash screen
2. Login/Register screen with simulated authentication
3. Main app with bottom navigation
4. Dashboard shortcuts for crowd monitoring, QR check-in, and notifications

## Screens Included

- Splash Screen
- Login/Register
- Dashboard/Home
- Schedule Manager
- Task Tracker
- Campus Map
- Crowd Monitoring
- QR Check-In
- Notifications
- Profile & Settings

## Notes

The app uses dummy data and local-only interactions. It is intended as a presentation-ready prototype that can be extended with Firebase, Room, Google Maps, QR scanning, or real DLSU data later.
